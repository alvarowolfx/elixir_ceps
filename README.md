# Elixir Ceps

Exilir Experiment to read a list of Brazilian Zipcodes from a file, do a request to an API on each zicode and the results for each zicode locally on a file.

## Running

```
mix escript.build
./elixir_ceps
```

Takes around 2~3 seconds to run.
